## 修改 main.c

在 main.c 中添加以下代码：

```c
#include "db_interface.h"
#include "tcp_server.h"

// 在 system_init() 中添加：
int system_init(void) {
    // ... 原有代码 ...
    
    // 初始化数据库接口（仿真模式）
    if (db_interface_init("radar_simulation.db") != 0) {
        printf("数据库初始化失败\n");
        return -1;
    }
    
    // 初始化TCP服务器
    if (tcp_server_init(5000) != 0) {
        printf("TCP服务器初始化失败\n");
        return -1;
    }
    tcp_server_start();
    
    // ... 原有代码 ...
}

// 修改 system_main_loop()，添加数据发送：
void system_main_loop(void) {
    while (1) {
        if (system_status.state == SYSTEM_RUNNING) {
            if (data_ready_flag) {
                // 仿真模式：从数据库读取数据
                db_read_wind_field(&wind_field_data);
                
                // ... 原有处理代码 ...
                
                // 通过TCP发送给上位机
                if (tcp_has_client()) {
                    // 构建并发送风场数据
                    // ...
                }
                
                data_ready_flag = false;
            }
        }
    }
}
```

## 修改 Makefile

添加新的源文件和sqlite3库：

```makefile
OBJS = main.o driver.o signal_processing.o data_transfer.o application.o \
       db_interface.o tcp_server.o

LDFLAGS += -lsqlite3 -lpthread
```

## 编译运行

```bash
# 编译
make

# 先运行数据库创建工具（生成radar_simulation.db）
./db_creator

# 运行雷达程序
./radar
```

## 上位机连接

1. 运行雷达程序后，TCP服务器监听端口5000
2. 上位机输入雷达IP和端口5000
3. 点击连接
4. 发送启动测量命令
5. 接收风场数据
